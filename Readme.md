## [Overview]()

This repository is the **main** hub for the 5th semester course Distributed Systems - Operational  Specialized Consulting to Support the Integrated Project) from the Humanistic AI and Data Science program at PUC-SP, taught by Professor Carlos Eduardo Paes.

Here you will find an organized view of all weekly classes, links to sub-repositories, study materials, and references that will guide you through the entire semester in a simple way.


[What You Will Learn (In Simple Words)]()

The course is about how many computers can **work together** as a team to solve problems faster and handle a lot of data.

You will learn what distributed systems are, how they talk to each other, and how to program them using modern tools like Python, Spark, Hadoop, and GPUs.
